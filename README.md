##  How to Play
###  Controls:

Accessible Controllers:
* Keyboard & Mouse, Xbox, PS4
* Player Controls/Actions:
  * Moving:
    * Keyboard: WASD
    * Xbox/PS4: Left stick / D-pad
  * Dropping an item:
    * Keyboard: Q
    * Xbox: LB
    * PS4: L1
  * Planting a seed:
    * Mouse: Right-click
    * Xbox: RT
    * PS4: R2
  * Punching:
    * Mouse: Left-click
    * Xbox: A
    * PS4: Cross
